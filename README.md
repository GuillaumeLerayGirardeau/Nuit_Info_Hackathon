# Application Nuit de l'Info – ESIEA

Cette application a été réalisée lors de la **Nuit de l'Info** à l'ESIEA.  
Elle est hébergée sur **Vercel** et distribuée sous licence **AGPL-3.0**.

## Thème
**Numérique, Inclusif, Responsable et Durable (NIRD)**

## Description
Cette application aide le public — élèves, enseignants, familles, collectivités — à comprendre comment un établissement peut réduire ses dépendances numériques et s’inscrire dans la démarche NIRD de manière progressive, réaliste et motivante.  

Elle propose également de petites activités ludiques pour découvrir et apprendre à utiliser des alternatives technologiques **open source, gratuites et libres de droit**.
